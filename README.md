# CORONA-SDK-Leviathan-GameEngine
What is Project Leviathan? Project Leviathan is Turned-based simulation Game Engine. This code is written by Lua, and run on Corona SDK. By using Corona SDK, you can publish this game as iPhone/iPad game, Android game, Amazon Fire, Windows or Mac Application.
